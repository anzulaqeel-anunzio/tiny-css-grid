# 10-Line CSS Grid System

A tiny, responsive CSS grid system powered by CSS Grid Layout.

## Features

-   **Super Lightweight**: Less than 150 bytes gzipped.
-   **Responsive**: Automatically stacks columns on small screens.
-   **Customizable**: Uses CSS variables for gap size and column width.

## Installation

1.  **Clone the repository** OR copy the CSS.
2.  **HTML Usage**:
    ```html
    <link rel="stylesheet" href="css/grid.css">
    ```

## Usage

```html
<div class="grid">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</div>
```

Customize via style attribute:
```html
<div class="grid" style="--min-width: 300px; --gap: 2rem;">
    ...
</div>
```
